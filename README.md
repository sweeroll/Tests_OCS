# OCS API tests


Use:
python 3.9/requests/pytest


### How to start
1. Use python 3.9 +
2. Create and activate virtual environments
    ```buildoutcfg
    python3 -m venv env
   source env/bin/activate
    ```
3. Run in terminal
   ```buildoutcfg
    pip install -r requirements.txt
    ```

4. Run tests from the folder **tests** with pytest or see **Makefile**

### Launch parameters
```
    --url      Base URL.
```
